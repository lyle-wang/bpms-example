"remoteintegration" is for those examples showing how to use remote BRMS resources, or how to connect to remote BRMS servers.
Before running any test in this folder, it is needed to setup / install BPMS 6.x and configure it for use.

[resources]: some resources will be used for setting up remote BRMS/BPMS server
[remotemaven]: showing how to use maven integration to download kjar from remote BPMS server and execute locally
[restapi]: showing how to connect to remote BPMS server to operate the kie runtime engine remotely
